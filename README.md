# [GPT Language Lion](https://virajsabhaya23-gptlanguagelion-appapp-dqf5z1.streamlit.app/)

## A translation bot that translates from one language to another

### To get API key follow the following steps and sign up 
1. Navigate to [https://platform.openai.com/signup](https://platform.openai.com/signup)
2. Navigate to [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)
3. click Create new secret key button and Copy the key 
4. Navigate to the [GPTlanguageLion](https://virajsabhaya23-gptlanguagelion-appapp-dqf5z1.streamlit.app/) → left menu → Paste the key
